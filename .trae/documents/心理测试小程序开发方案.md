# 心理测试H5项目 - 零成本实施方案

## 技术架构
- **前端**: Vue3 + Vant → 部署到 Vercel (免费)
- **后端**: Node.js → 部署到 阿里云函数计算 (免费额度内)
- **数据库**: MongoDB Atlas (免费512MB)
- **域名**: 使用平台自带子域名 (免费)

## 开发阶段

1. 初始化Node.js项目
2. 连接MongoDB Atlas
3. 实现兑换码验证接口
4. 实现答题提交接口
5. 实现报告生成接口 (规则算法)
